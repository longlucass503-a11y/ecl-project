### 关键验证链路示例

```
借据 AST_C01（对公LC·不动产·CRR3·逾期0天）
  → 风险分组: GRP_CORP（匹配优先级1）
  → 阶段判定: STAGE_1（兜底规则·无触发）
  → PD: [BASELINE×0.5=0.25% + UPTURN×0.2=0.05% + DOWNTURN×0.3=0.30%] = 0.60%
  → EAD: 还款折现 ¥94,392 + 表外 ¥51,300 = ¥145,692
  → LGD: 0.35（精确匹配GRP_CORP|不动产|LC）
  → ECL: 0.0060 × 0.35 × 145,692 = ¥306
  → Overlay: 全局+30BP = ¥145,692 × 0.003 = ¥437
  → ECL最终: ¥306 + ¥437 = ¥743

借据 AST_C02（对公LC·不动产·逾期45天·CRR5）
  → 风险分组: GRP_CORP（匹配优先级1）
  → 阶段判定: STAGE_2（逾期31-90天规则触发）
  → PD: 12M=1.55%, 存续期=1-(1-0.0155)^1.5=2.32%（到期日2027-12-31距计量日1.5年）
  → EAD: ¥478,125 + ¥59,400 = ¥537,525
  → LGD: 0.35
  → ECL: 0.0232 × 0.35 × 537,525 = ¥4,365
  → Overlay: 全局+30BP + 关注类FIXED¥5000 = ¥537,525×0.003 + ¥5,000 = ¥6,613
  → ECL最终: ¥4,365 + ¥6,613 = ¥10,978

借据 AST_C05（零售PL·房产抵押池·逾期30天·CRR4·CRR评级下降2级）
  → 风险分组: GRP_RETAIL（匹配优先级1）
  → 阶段判定: STAGE_2（CRR评级下降规则触发·递补逾期边界也可触发）
  → PD: 12M=0.80%, 存续期=1-(1-0.008)^3.54=2.79%（到期日2030-01-15）
  → EAD: ¥276,448 + ¥33,600 = ¥310,048
  → LGD: 池计算→押品净价值(¥400K×0.98×0.8+¥80K×0.9×0.7)=¥313,600+¥50,400=¥364,000 > EAD→全覆盖
           按LGD下限10%算
  → ECL: 0.0279 × 0.10 × 310,048 = ¥865
  → Overlay: 零售+50BP → ¥310,048 × 0.005 = ¥1,550
  → ECL最终: ¥865 + ¥1,550 = ¥2,415
```
